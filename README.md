# Thecnical stuff .. for nerds
I built moglinTV by composing him of 3 parts, (i know you didnt sign up for moglin atonomy 101 but here we are) a Webserver Docker  container hosting his faces and charpage proxy all built in python using flask,
A hardware contianer handling the logic of the tail servo, and acceleromter, and a client to interact with AQW via machine vision by using edge detection and template matching. All parts communicate via http between eachoter.

diagram of communication

## The Moglin guts :/
- The moglins main brain is a sick Raspberry pi 5 8gb of ram 
- His face is a 5 inch screen
- his tail is controlled by a sg90 servo
- A adxl335 Acceleromter connected to an ads1115 analog to digital converter
all "neatly" solder to a prototype board. and put inside a nice 3D printed moglin skinsuit.
